# Antigravity Analytics MVP 🚀

Профессиональная система финансовой аналитики и P&L-отчетности для мультиселлерных аккаунтов на маркетплейсах (Wildberries & Ozon).

## 🌟 Основные возможности

- **Мультиселлерность**: Автоматическая обработка данных для неограниченного числа кабинетов (ООО/ИП).
- **Автоматизация ETL**: Полный цикл «Google Drive -> Python -> Google Sheets».
- **Умный P&L**: Расчет чистой прибыли, маржинальности, ROI и учет логистики/хранения/удержаний.
- **AI-Инсайты**: Генерация бизнес-аналитики с помощью GigaChat (автоматические выводы и рекомендации).
- **Автоопределение периода**: Система сама находит даты в отчетах, минимизируя ручной ввод.
- **Облачный запуск**: Настроенный GitHub Actions для работы по расписанию.

## 🛠 Технологический стек

- **Core**: Python 3.12
- **Data**: Pandas, NumPy, PyArrow
- **APIs**: Google Drive API v3, Google Sheets API v4, GigaChat API
- **CI/CD**: GitHub Actions, Docker

## 🚀 Быстрый старт (Настройка в облаке)

Чтобы система заработала автоматически на GitHub:

1. **GitHub Secrets**: Перейдите в `Settings -> Secrets and variables -> Actions` и добавьте:
   - `GOOGLE_CREDENTIALS_JSON`: Содержимое вашего файла `credentials.json`.
   - `GOOGLE_DRIVE_ROOT_FOLDER_ID`: ID корневой папки на Google Drive.
   - `GIGACHAT_AUTH_KEY`: Ваш ключ авторизации GigaChat.
   - `GIGACHAT_SCOPE`: Область действия API (например, `GIGACHAT_API_PERS`).

2. **Запуск**: Система будет запускаться каждый понедельник автоматически. Для ручного запуска перейдите во вкладку `Actions`, выберите `Antigravity ETL Pipeline` и нажмите `Run workflow`.

## 📂 Структура проекта

- `main.py`: Точка входа и управление пайплайном.
- `consolidate.py`: Ядро финансовых расчетов.
- `drive_utils.py`: Модуль взаимодействия с Google Drive.
- `sheets_utils.py`: Модуль записи данных в Google Sheets.
- `ai_analyzer.py`: Интеграция с нейросетью GigaChat.
- `.github/workflows/`: Настройки автоматизации.

## 🔒 Безопасность

Проект настроен на работу через переменные окружения и секреты. Конфиденциальные данные (`.env`, `credentials.json`, кэши) защищены файлом `.gitignore` и никогда не попадут в репозиторий.
